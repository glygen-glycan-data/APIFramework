GlyLookup



Input: Glycan in GlycoCT or WURCS sequence.
Output: GlyTouCan accession of sematically the same glycan.
GlyLookup takes your glycan as input, and run exact alignment algorithm on all GlyTouCan glycan to find out the accession.
